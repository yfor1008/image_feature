# 线检测

## 线的定义

由一系列连续的点构成的线, 具有一定的方向性.

## 线检测原理

使用带方向性的滤波核对图像进行空间滤波, 计算公式与点检测相同. 常用的滤波核为:

- 水平方向

|      |      |      |
| :--: | :--: | :--: |
|  -1  |  -1  |  -1  |
|  4   |  4   |  4   |
|  -1  |  -1  |  -1  |

- 垂直方向

|      |      |      |
| :--: | :--: | :--: |
|  -1  |  4   |  -1  |
|  -1  |  4   |  -1  |
|  -1  |  4   |  -1  |

- 45°方向

|      |      |      |
| :--: | :--: | :--: |
|  -1  |  -1  |  4   |
|  -1  |  4   |  -1  |
|  4   |  -1  |  -1  |

- 135°方向

|      |      |      |
| :--: | :--: | :--: |
|  4   |  -1  |  -1  |
|  -1  |  4   |  -1  |
|  -1  |  -1  |  4   |
